# أشرقت ❤️

موقع حب وتحفيز مذاكرة مخصص لأشرقت

## المميزات ✨

### 💕 رسائل الحب
- 13 رسالة حب جميلة
- تنطق صوتياً باللهجة المصرية

### 🎮 الألعاب
1. **لعبة الذاكرة** - اختبري الذاكرة مع رسائل حب
2. **دوار القلوب** - اختبري الحظ والحب مع 8 رسائل حب
3. **لعبة الوقت الجميل** - اختاري الوقت المفضل وأجيبي على أسئلة

### 📚 وضع المذاكرة الذكي
- ✅ إضافة مواد يومية
- ✅ عداد ذكي (50 دقيقة للمادة)
- ✅ 15 رسالة تحفيز كل 10 دقايق
- ✅ تنبيهات ذكية (نصف المدة، 5 دقايق، آخر دقيقة)
- ✅ استراحة ذكية مع نكت ورسائل حب
- ✅ نظام النجوم والإنجازات
- ✅ 🔥 عداد أيام متتالية (Streak)
- ✅ إحصائيات مفصلة (أسبوعي، شهري، أفضل المواد)

### 🌙 وضع ليل/نهار
- Dark mode (افتراضي)
- Light mode

### 📱 PWA (Progressive Web App)
- يعمل بدون انترنت (Offline)
- يمكن حفظه على الهاتف كتطبيق
- سريع جداً

## الاستخدام 🚀

### تشغيل محلياً
```bash
# استخدم أي web server
python -m http.server 8000

# أو
npx http-server
```

ثم فتح: `http://localhost:8000`

### نشر على Vercel
```bash
vercel
```

## التكنولوجيا 💻
- HTML5
- CSS3 (Responsive)
- Vanilla JavaScript
- Service Worker (PWA)
- Canvas API (الورد المتساقط)
- Web Speech API (Text-to-Speech)
- LocalStorage (حفظ البيانات)

## الملفات 📂
- `index.html` - الصفحة الرئيسية
- `style.css` - التصميم والأنماط
- `app.js` - البرمجة والوظائف
- `sw.js` - Service Worker
- `manifest.json` - PWA Configuration
- `vercel.json` - Vercel Configuration
